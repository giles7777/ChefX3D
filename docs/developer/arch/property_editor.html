Property Editor Design

Each entity has a set of properties sheets associated with it. These properties are a parameter space for some domain like simulation parmaters, behavior parameters or visual properties.

Concepts

Parameter Space Declaration

ChefX3D uses an XML schema and an instance document to describe a parameter space, its data types, and its intial values. The schema gives a hierarchical structure, parameter names and data types. The instance document gives the initial values. The instance can be tool specific. A property editor will turn this data into an editable form.

Current implementation of this only uses an instance document. Future implementations will use a schema for further information.

Undo/Redo

A model shall be able to undo/redo changes. The command list will be a global list of changes made to the model.

Export

These property sheets will become the XML space over which exporters will work. In the current implementation this is implemented using a XML Stylesheet. ChefX3D will add some tool parameters to this. Some of those parameters are the entities location and orientation.

Data Editors

Any parameter can be assigned a data editor. A data editor is a custom way of editing that parameter or an extra feature to go with it. An example might be a parameter representing a file name might include a FileDialogEditor to open a file dialog box to fill in its value.

User could should be allowed to create their own custom editors. Editor's will have an API to modify the current property tree including adding or deleting properties.

Editors provided by the system include:

  • File Dialog - Use a File Dialog to select a file name
  • Combo Box - Use a combo box to select a value
  • Increment Number - Increment a number field
  • Decrement Number - Decrement a number field
  • Add Property - Add more properties to this instance
  • Delete Property - Delete properties from this instance
  • Add an Association - Create an association with another entity
  • Delete an Association - Delete an association with another entity
